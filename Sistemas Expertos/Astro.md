### Comandos

Crear un proyecto en astro: 
```bash
npm create astro@latest 
```

Subirlo a producción es: 
```bash
npm run build (la carpeta dist estan los archivos)
```

Obtener los comandos: 
```bash
npx astro add --help
```

Levantar el servidor: 
```bash
npm run dev
```
